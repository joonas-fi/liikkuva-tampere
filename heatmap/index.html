Lukuunottamatta tiettyjä erikoistapauksia ( itsenäisyyspäivä jne. ), pysyy liikenne vuoden aikana ( 11.8.2014 - 31.5.2015 ) hyvin samankaltaisena. Tästä syystä pysäkkien aktiivisuusaste-animaatio generoidaan vain viikon ajalta.
Tämä näkymä tarjoaa mahdollisuuden tarkastella Tampereen alueen linja-autopysäkkien aktiivisuusasteita eri kellonaikoina.
